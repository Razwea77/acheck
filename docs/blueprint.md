# **App Name**: AutoCheck24 заявка

## Core Features:

- Display Customer Data: Displays customer name, diagnostic cost, request status, and request ID.
- Display Car Data: Displays car brand, model, and contact number. These should be dynamically updatable via Telegram Bot.
- Confirm Request Button: A button to confirm the request, placed at the bottom with styling matching the screenshot.

## Style Guidelines:

- Primary color: Teal-ish Blue (#45A7B1) for main elements, resembling the neon glow effect.
- Background color: Dark navy blue (#1A2B3D) to create a dark mode appearance.
- Accent color: Cyan (#58D1C9) for interactive elements and highlights to complement the primary color and create a futuristic feel.
- Body and headline font: 'Inter' sans-serif font for clear and modern readability.
- Data displayed in a slightly glowing dark square container.
- Use simple, outline-style icons for each data category (customer, car, etc.).
